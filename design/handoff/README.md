# Handoff: BabyLog — 육아 슈퍼앱 (전체 프로토타입)

> 개발 대상: **Flutter (iOS 우선, Android 동시 지원)**
> 디자인 피델리티: **High-fidelity** — 최종 컬러·타이포·간격·인터랙션 포함

---

## 1. Overview

BabyLog는 **임신 → 출산 → 육아**로 이어지는 양육 여정을 하나의 앱에 담은 동네 기반 육아 슈퍼앱입니다. 핵심은 다섯 축입니다.

- **기록** (임신·태아 일지 + 아이 성장 기록·예방접종)
- **동네** (주변 인프라/응급 + 중고 마켓 + 육아 크루)
- **가계부** (육아비 + 정부지원금)
- **뱃지·신뢰도** 시스템 (TickLab 계보)
- **프라이버시·신뢰** 원칙 (데이터 비매각, 무광고)

이 번들의 HTML 파일은 **디자인 레퍼런스(프로토타입)** 입니다. 그대로 배포하는 코드가 아니라, **의도된 화면과 동작을 보여주는 참조물**입니다. 작업 목표는 이 디자인을 **Flutter 프로젝트에 재구현**하는 것입니다. 기존 코드베이스가 있다면 그 패턴(상태관리, 디자인 토큰, 위젯 구조)을 따르고, 없다면 아래 권장 구조로 시작하세요.

> **언어/톤:** 모든 UI 카피는 한국어, 따뜻하면서 단정한 존댓말. 성별 중립("양육자/○○님"), 상실·이혼·입양 등 엣지 케이스 배려 카피 원칙을 유지하세요.

### 3.4 시그니처 효과 — "물 흐르는" 리퀴드 버튼 (`.bl-liquid`)
주요 채움 CTA(저장·전화·시작 등)에 적용. 두 레이어:
1. **광택 메니스커스**(항상): 상단 radial 하이라이트 + 미세 음영. `radial-gradient(130% 70% at 50% -16%, rgba(255,255,255,.34), transparent 62%)`.
2. **흐르는 빛 띠**(애니메이션): 폭 50%의 흰 그라데이션 띠가 `-50%→128%`로 사선(skewX -14°, blur 3px) 이동, 4.6s 루프. 누르면 1.1s로 가속. `prefers-reduced-motion`에서 비활성.
Flutter: `AnimatedBuilder` + `ShaderMask`/`Stack` 오버레이로 구현. reduced-motion은 `MediaQuery.disableAnimations` 체크.

## 4. 정보구조 & 네비게이션 (기능 8 — 확정 사항)

**하단 5탭 (균등 배치, 중앙 FAB 아님):**
`홈 · 기록 · 동네 · 가계부 · 내정보`

- 탭바: 높이 + safe-area. iOS 하단 inset ~26, Android ~14. 반투명 흰 배경 + blur, 상단 1px 라인. 아이콘 23px, 라벨 10px. 활성=primary, 비활성=ink-3.
- **빠른 기록 FAB:** 화면 **우하단** 플로팅(좌하단 이동 옵션 — 한 손 조작 접근성). 탭바 위로 띄움. 홈·기록·동네 탭에서만 노출. 탭하면 스피드다이얼(사진/성장측정/메모, 임신모드는 태동/배사진/메모)이 위로 펼쳐지고 FAB는 45° 회전. **중앙 탭바 버튼은 채택 안 함**(엄지 동선).
- **동네 탭**은 내부 세그먼트 `주변 / 마켓 / 크루`. 헤더 우측 액션이 세그먼트별로 변함(주변→응급, 마켓→팔기, 크루→모임 만들기). 시간대 적응 노출(밤=소아과 우선, 낮=마켓 우선) 권장.
- **딥링크:** 홈 카드/알림 → 해당 전용 탭의 상세로 직접 이동. 탭 전환 시 스크롤·필터 상태 보존. 뒤로가기 일관성 유지.

**상태 모델(권장):** `tab`, `mode('baby'|'pregnancy')`, `dongneSeg`, `recordSeg`, `childId`, `fabSide`, push stack, sheet. (app/shell.jsx의 `Root`/`AppContent` 참조 — Flutter에서는 `Navigator` + `IndexedStack`(탭 상태 보존) + Riverpod/Bloc 권장.)

## 5. 화면별 명세

> 각 화면의 정확한 픽셀·카피는 `app/<file>.jsx`에 있습니다. 아래는 레이아웃·동작 요약.

### 5.1 온보딩 — `app/onboarding.jsx`
스플래시("우리 동네 육아의 모든 것") → **게스트 가치 미리보기**(가입 전 야간 소아과 즉시 표시) → 기록 밀도 선택(가볍게/꼼꼼히) → **임신/출산 분기**(🤰임신중 / 👶출산했어요) + 이름·예정일/생일 입력(입력 시 주차·접종 타임라인 자동 생성 안내) → 프리퍼미션(위치/알림, 맥락 설명 후 요청). 모든 단계 스킵 가능, 강제 입력 0. 상단 진행바.

### 5.2 홈 — `app/home.jsx` (육아중 3 레이아웃 + 임신 전용)
세션 첫 화면 = "오늘의 한 장면". 상단 인사 + 우측 **응급** 버튼 + 다자녀 전환 칩.
- **우선순위 엔진 단일 카드**(지금 가장 중요한 것 하나만 크게: 접종 D-1, 지원금 마감, 응급, 1년 전 오늘 등)
- 하위 모듈 3~4개(또래 이야기, 추억 "1년 전 오늘", 동네 소식, 가계부 요약, 기록 권유) — 카드당 단일 액션
- **레이아웃 3안**(컨트롤바 토글): A 히어로(최근 사진 풀블리드 헤더) / B 대시보드(타일 그리드) / C 타임라인(최근 기록 인라인)
- **임신 모드 홈**: 태아 히어로(주차·과일·D-day) + 검진 우선카드 + 주차 가이드 + 동네/가계부 모듈

### 5.3 기록 탭 — `app/record.jsx` (육아) / `app/pregnancy.jsx` (임신)
**육아:** 세그먼트 `타임라인 / 성장차트 / 예방접종`.
- 타임라인: 날짜 그룹, 사진·성장측정·접종 카드. 이정표 태그(앰버 뱃지).
- 성장차트: WHO 곡선 오버레이 SVG(p15/p50/p85 밴드 + 실측 라인), 상단 **정성적 안심 메시지**("또래와 비슷하게 잘 크고 있어요"), 백분위는 펼쳐서. 키/몸무게/머리둘레 토글.
- 예방접종: 질병관리청 스케줄 기반, D-7/D-1/당일 알림, 완료 체크.
- 헤더 우측 **공유 아이콘 → 성장 카드 에디터**.

**임신(`pregnancy.jsx`):** 태아 히어로(튼튼이 24주 3일, 옥수수 🌽 비유, 출산 D-day) + 세그먼트 `태아 가이드 / 산모 기록 / 산전 검사`.
- 태아 가이드: 주차별 발달(키/몸무게/과일), 지난 주차 타임라인.
- 산모 기록: **태동 카운터**(10회 목표 도트), 체중 추이 차트(권장 증가 밴드), **배 사진 D라인** 가로 타임라인(출산 후 성장 사진과 연속), 컨디션 태그.
- 산전 검사: 임당검사·정밀초음파 등 D-day.
- 우상단 **"출산했어요" → 출산 전환 플로우**: 아이 이름·실제 생일 입력 → "세상에 온 걸 환영해요" → 배사진→성장사진 연속, 태명→이름, 모드 baby 전환. 임신 기록은 '태아 시절' 아카이브로 영구 보존.

### 5.4 빠른 기록 시트 — `app/record.jsx` `QuickRecordSheet`
FAB에서 호출. **초경량(기본):** 사진 드롭존 + 이정표 칩 + 저장(2탭 완료). **자세히 펼치기:** 메모·키·몸무게 + "AI 캡션 초안(Pro)". 저장 후 보상 애니메이션("○○이의 153번째 순간"). 바텀시트, 드래그 핸들, slideUp.

### 5.5 동네 — 주변/응급 `app/nearby.jsx`
세그먼트 주변. 카테고리(소아과/약국/키즈카페/놀이터) + 필터 칩(현재 영업중/야간진료/공휴일…). **리스트 퍼스트**(지도 토글 제공). 병원 카드: 영업상태 뱃지, 거리/평점, **"○분 전 확인" 신뢰도 뱃지**, 큰 전화 버튼. 상세: 전화/길찾기/즐겨찾기, 영업표, "지금 문 닫혔어요" 신고, 면책 고지.
- **응급 모드**(별도 다크 풀스크린): "지금 갈 수 있는 곳" 즉시(필터 0), 현재 영업중·거리순, **초대형 전화 버튼**(`#FF5C42`), 마지막 확인 시각, 119 상담. 고대비·큰 터치타깃·최소 정보(새벽·불안한 부모 최적화).

### 5.6 동네 — 마켓 `app/market.jsx`
세그먼트 마켓. **"곧 필요해요"** 월령 연동 추천 가로 피드. 카테고리 칩. 매물 카드: 사진, 상태등급(S/A/B/C 컬러 뱃지), 월령 태그, **리콜 경고**(coral), **졸업템** 표시, 판매자 미니 뱃지(티어+신뢰), 거리/관심. 상세: 판매자 카드, 리콜 경고 박스, 위생 셀프체크, 채팅 CTA. 판매(성장 졸업 트리거): 사진 → **AI 자동 분류**(온디바이스) → 등급·시세 제안 → 등록. 채팅: 앱 내, 안심 거래존 안내.

### 5.7 동네 — 크루 `app/crew.jsx`
세그먼트 크루. **같이 가요**(모임: 장소·일시·정원, 참가), **비슷한 또래 크루**(반경·월령·관심사 태그), **동네 게시판**(정보공유/같이해요/고민상담). **콜드스타트 빈 상태 = 기대감 UI**(컨트롤바 크루:오픈전): "우리 동네 78% 준비됨" 진행바 + 대기명단 + 친구 초대 + 초기멤버 혜택. 밀도 미달 시 기능 숨김(피처 플래그). 전화번호 인증·신고 누적 자동제재.

### 5.8 가계부 — `app/budget.jsx` (독립 탭)
**정부지원금 전면 배치**(아동수당 D-4 등 — 신청방법/서류/복지로 딥링크). 도넛 차트 + 카테고리 분해. 월령별 예상 지출 가이드. **자동 수집 거래**(마켓·구독 자동 기록, "자동" 뱃지) — 큰 지출만 수동 추가. 부부 공유. 우하단 add-expense FAB.

### 5.9 내정보 — `app/profile.jsx`
프로필 카드(티어 뱃지 + 보조 뱃지 최대 3 + 통계 4분할). **골든맘까지** 티어 진행바. **Pro 업셀**(다크 카드). **뱃지 컬렉션 그리드**(획득=컬러, 미획득=잠금 — 수집 욕구). **데이터·프라이버시 절대원칙** 행(가족 공유, 데이터 비매각, 내보내기, 양육자 역할 중립). "광고 없음·비매각·영구 보존" 고지. Pro 화면: 기능 리스트 + 월/연 요금(3,900/29,000) + 7일 무료.

### 5.10 성장 카드 공유 — `app/sharecard.jsx`
인스타용 바이럴 카드 에디터(다크). 사진 위 데이터 오버레이 + 하단 스크림 + 우상단 BabyLog 워터마크. 컨트롤: **비율**(4:5/1:1/9:16), **데이터 위치**(좌하/우하/좌상/중하/없음), **표시 데이터 토글**(키/몸무게/월령·D+day/또래백분위/이정표), **프라이버시**(얼굴 가리기 블러, 워터마크 토글 — 무료=포함, Pro=제거). Flutter: `RepaintBoundary` + `toImage()`로 카드 렌더 후 `share_plus`.

### 5.11 뱃지·티어 시스템 — `00 Design System.html`, `app/data.jsx`
4티어: 새싹(grey) → 따뜻한 이웃(mint) → 믿음직한 맘(purple) → 골든 맘(amber/gold). 티어 분기 1회 자동 재산정(최근 6개월), 활동 뱃지 즉시 부여, 특별 뱃지(베타·초기멤버) 영구. 신규 획득 시 푸시 + 축하 애니메이션. 노출 규칙: 프로필=티어1+보조3, 매물=2, 채팅=티어1.

## 6. 인터랙션 & 동작

## 8. 기술 스택 메모 (스펙 권장)

스펙은 SwiftUI MVP를 가정하나 **본 프로젝트는 Flutter**입니다. 매핑:
- 로컬/동기화: CoreData+CloudKit → **Drift/Isar + 가족 공유 백엔드**
- 백엔드(마켓 이후): **Supabase**
- 결제: StoreKit2 → **in_app_purchase**
- 차트: Swift Charts → **fl_chart / 커스텀 CustomPainter**(성장곡선·도넛·체중추이는 SVG 로직을 CustomPainter로 이식)
- 위젯/워치: WidgetKit → 홈 위젯(오늘 할 일·아이 요약·주변 응급)
- AI: 온디바이스 비전(무료) + 서버 LLM(Pro 캡션). 의료 Q&A는 직접 답변 보류, 출처 큐레이션 대체.
- 외부 API: 건강보험심사평가원, 카카오맵, 질병관리청 예방접종, 복지로, 카시트 리콜 DB.
- 원격 구성(피처 플래그)로 동네별 점진 개방.

**절대 원칙(타협 불가):** 데이터 비매각 · 무료 데이터 영구 보존 · 무광고 · 정직한 결제(다크패턴 금지) · 성별 중립 · 아동 안전 · 안정성 우선 · 데이터 주권(상시 내보내기).

## 9. Assets

- **폰트:** Pretendard (Variable) — pretendard 패키지 또는 폰트 파일 번들. 라틴 숫자 tabular feature.
